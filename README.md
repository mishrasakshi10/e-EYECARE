# e-EYECARE
👁️ e-EYECARE: Online Eye Clinic System

📌 Project Overview

e-EYECARE is a web-based Eye Clinic Management System developed to simplify the process of booking eye care appointments and managing patient records online. The system enables patients, doctors, and administrators to perform their respective tasks through a secure and user-friendly platform.

The project aims to reduce manual paperwork and provide an efficient digital solution for eye clinic management.

✨ Features

👨‍⚕️ Patient Module
Patient registration and login
Profile management
Book appointments

🩺 Doctor Module

🚀 Installation

Clone the repository.
Import the project into Eclipse IDE.
Configure Apache Tomcat Server.
Create the MySQL database.
Import the SQL database file.
Update database credentials in the project.
Run the project on Tomcat.
Open the application in your browser.
